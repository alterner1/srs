Вычислитель родственных связей.

Входные данные:
Набор объектов, обладающих именем (уникальным) и полом (М или Ж)
Два типа связей: супруг<->супруг, родитель->потомок

Выходные данные:
Список всех родственников для заданного имени

Языки:
LUA
Prolog
SQL
Rust
Python
Go
SCALA

Петя М
Маша Ж
Петя <-> Маша
Коля М
Петя -> Коля

? Коля:
Маша -> Коля - сын
Петя -> Коля - сын
